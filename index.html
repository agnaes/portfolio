eVisa India – UX/UI Case Study

Project Overview

eVisa India is a conceptual redesign project focused on creating a user-friendly, modern web experience for individuals seeking various types of Indian visas. Developed as part of a recruitment task, the project simulates a real-world challenge: simplifying a complex bureaucratic process through clear structure, smart content design, and strong visual hierarchy.


Project Type: Recruitment Design Task
Role: UX/UI Designer
Platform: Web
Tools Used: Figma, Photoshop, Illustrator (for images and icons)
Target Audience: Travelers applying for an Indian visa online


Design Objectives

  • Simplify the visa selection process
  • Build trust through clear communication and visual consistency
  • Make navigation intuitive even for users unfamiliar with digital forms
  • Ensure full responsiveness across devices

Information Architecture & UX Strategy

The core UX challenge was structuring a system that supports quick understanding and confident action.

Key decisions included:

  • User-centered categorization:Visa types are organized based on user intent (e.g., Tourist Visa, Business Visa, Medical Visa) instead of formal legal definitions, making navigation intuitive and reducing decision fatigue.
  • Clear entry points: Users are guided from the homepage directly into their most likely use case via smartly labeled action buttons and visual cards.
  • Content hierarchy: Each section follows a consistent pattern— Do I need an eVisa?→ How does it work? → Which type of visa do I need? → What is the waiting time? → Apply now. This reduces cognitive load and sets user expectations.

Open Sans, sans-serif

The font choice was due to keeping balance between legibility, neutrality, and accessibility — all of which are crucial in a web service that deals with sensitive, informational content like visa applications.

Conceptual UI Design (Smartwatch App)

Project Overview

smartwatch interface designed to help users track and manage key aspects of their personal well-being and device performance — all within a minimal, elegant interface tailored for a small screen. The goal was to design a clean, glanceable UI that delivers maximum information with minimal friction - a fundamental challenge in wearable UX design.


Role: UX/UI Designer
Platform: Application
Tools Used: Figma, Photoshop, Illustrator (for images and icons)


Design Objectives

  • Designed for Glanceability - In smartwatch UX, screen time is brief — users interact in bursts of 3–5 seconds. The interface prioritizes iconography, bold typography, and color-coded signals to deliver information quickly without requiring deep navigation.
  • Touch-Friendly Layout - All actionable elements are large enough to be easily tapped on small screens, with spacing optimized for thumb interaction — no pinching or accidental taps.
  • Data Hierarchy & Focus- Each screen follows a single-purpose structure, removing clutter and focusing the user’s attention:
    Home: Activity & wellness snapshot
    My Device: Tech and connectivity
    My Profile: Personal data only

Roboto

Roboto was chosen for its excellent legibility on small screens, modern aesthetic, and optimized performance for digital interfaces—making it ideal for a smartwatch environment

Application UI Design

UX/UI Case Study

Project Overview

Mobile application designed for a modern beauty salon, allowing users to create an account, book appointments with selected beauticians, browse services, read customer reviews, and explore current promotions. The app enhances the client experience by combining functionality with elegance, making self-care easy, intuitive, and personalized.


Role: UX/UI Designer
Platform: Application
Tools Used: Figma, Photoshop, Illustrator (for images and icons)

Design Objectives

  • Simplify the booking process - through a clear and guided user flow
  • Present services visually - allowing users to easily choose from options like manicure, eyelash extensions, skincare, and more
  • Build credibility- with an accessible reviews section, showcasing ratings and feedback
  • Simplify the booking process - via dynamic promotional banners highlighting current deals
  • Create a polished, welcoming interface - that reflects the aesthetics and professionalism of a high-end beauty salon

Helvetica

Helvetica was chosen for its timeless elegance, clean geometry, and high legibility—perfectly aligning with the app’s beauty-focused brand and ensuring clarity across both headings and body text.

UX Case Study – Medical Staff Scheduler Interface

Project Overview

This interface was created as part of a recruitment design task and focuses on building a scheduling tool for medical staff responsible for patient care. The main goal was to design a clear, scalable dashboard that supports daily task planning while providing quick access to essential information about patients, assignments, and caregivers.


Project Type: Recruitment Design Task
Role: UX/UI Designer
Platform: Web
Tools Used: Figma, Photoshop, Illustrator (for images and icons)


Design Objectives

  • Streamline navigationacross complex datasets: patients, staff, events, and devices
  • Provide a clean, intuitive scheduling systemwith detailed views upon selecting a patient
  • Minimize cognitive load through a clear visual hierarchy, functional color usage, and logical layout
  • Create a modern, professional interfacealigned with the standards of medical and enterprise systems

Interaction Flow & Key UX Elements

  • Clicking on a patient’s name opens a detailed individual schedule view, organized by date and time.
  • Left Sidebar Menu Includes quick access to key modules: Map, Events, Caregivers, Devices, Messages Designed to support multiple user roles (e.g. coordinators, field workers, administrators).
  • Top Navigation Bar Main categories: Portal, Operations, Administration These provide contextual access to broader platform features, separating operational use from system management.

Montserrat

Montserrat was selected for its modern, geometric structure and excellent readability in data-dense interfaces. Its clean lines and versatility make it ideal for dashboards, providing both a professional appearance and user-friendly legibility across table content, menus, and forms.
